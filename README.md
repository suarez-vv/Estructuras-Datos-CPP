# Estructuras-Datos-Y-POO-C-CPP

Colección de prácticas y proyectos desarrollados en C++ enfocados en estructuras de datos, recursividad, árboles, grafos, tablas hash y conceptos fundamentales de clases.

El repositorio reúne implementaciones académicas que muestran el aprendizaje progresivo de estructuras lineales y no lineales, algoritmos de recorrido, búsqueda y aplicaciones construidas a partir de dichas estructuras.

## Objetivo

Practicar la implementación y aplicación de estructuras de datos y conceptos de clases en C++, mediante el desarrollo de algoritmos, recorridos, modelos de almacenamiento y proyectos que permitan comprender su funcionamiento interno.

## Prácticas Destacadas

### Árboles Binarios de Búsqueda

Implementación de árboles de búsqueda para operaciones de inserción, recorrido y consulta de información.

### Heaps y Heap Sort

Implementación de estructuras Heap y algoritmos de ordenamiento basados en prioridades.

### Grafos

Desarrollo de representaciones mediante listas y matrices, así como algoritmos de recorrido y búsqueda de caminos.

### Dijkstra

Implementación de variantes del algoritmo de Dijkstra para grafos ponderados y no ponderados.

### Tablas Hash

Implementación de estructuras de almacenamiento basadas en funciones hash para optimizar búsquedas.

Aplicación de colas, árboles binarios y recursividad para modelar y simular torneos deportivos.
